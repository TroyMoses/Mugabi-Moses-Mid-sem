WHAT IS CSS?

Css stands for Cascading Style Sheets. But what is it exactly!?

Well, this is special or unique markup language that is used in web development in combination with hypertext markup language to style and design the presentation of a web page or many of them. There are mainly three types of css, namely;

  1. Internal css: This works mainly within the particular web page to be styled. It is used to style the different components of the web page within the same web page.
  2. Inline css: This works like the internal css, but the difference is that for the inline css, we style a particular element at a go but not all the related elements within the web page. Simply we style an element using its corresponding attributes.
  3. External css: An external css is a separate and independent css file that is used to style the presentation and appearance of a web page as well as different same web pages to be styled in the same format. This external css file though, can be accessed by creating a link to it in the web page to be styled.

Examples on how to insert css

  1. Internal css: This is used in the head tag
  2. Inline css: This used as an attribute of an element.
  3. External css: This is used in a different css file created and linked from the page to be styled.

CSS SELECTORS.

These are used in css to identify a particular element to be styled. Here are a few of them;

  1. Class selectors: These select or identify a number of elements to be styled at once.
  2. Id selectors: These identify only a single unique element per web page.
  3. Universal selectors: These style a particular element to be applied to all the web pages.
